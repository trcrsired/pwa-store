📦 如何安装渐进式网络应用（PWA）

PWA 应用可以直接通过浏览器安装，提供接近原生应用的体验。以下是各主流平台的安装方法：

🧭 Chrome（Windows、macOS、Linux、Android）

  • 访问 PWA 网站。
  • 在地址栏中查找 安装图标（一个带箭头的显示器）。
  • 点击 安装，即可添加到桌面或主屏幕。
  • 在 Android 上，点击 三点菜单 → 安装应用。

🦊 Firefox

  • Firefox 当前不支持原生 PWA 安装。
  • 可使用 PWAsForFirefox 扩展插件实现桌面支持。

🧑‍💻 Edge（Windows、macOS、Android）

  • 访问 PWA 网站。
  • 点击地址栏中的 安装图标，或进入 设置 → 应用 → 将此网站安装为应用。
  • Android 上的安装方式与 Chrome 相同。

🍎 Safari（iOS、macOS）

  • 在 iOS 上：点击 分享 图标 → 添加到主屏幕。
  • macOS 上：Safari 不支持原生 PWA 安装。
  • 建议使用 Chrome 或 Edge 浏览器以获得完整支持。

📱 Android（所有浏览器）

  • 大多数现代浏览器均支持 PWA 安装。
  • 在浏览器菜单中查找 安装 或 添加到主屏幕 选项。

🧩 技术提示

  • PWA 必须通过 HTTPS 提供服务，并包含有效的 manifest.json 文件。
  • 可通过 Chrome DevTools → Application 标签页测试可安装性。
  • 安装后的 PWA 可像原生应用一样从操作系统中启动。

想了解更多？请访问 Google 的 PWA 指南。
