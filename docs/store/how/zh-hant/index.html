📦 如何安裝漸進式網頁應用程式（PWA）

PWA 應用可直接以瀏覽器安裝、提供接近原生應用之體驗。以下爲各主流平臺之安裝方法：

🧭 Chrome（Windows・macOS・Linux・Android）

  • 訪問 PWA 網站。
  • 于地址欄中査找 安裝圖示（一帶箭頭之顯示器）。
  • 點擊 安裝、即可添加至桌面或主熒幕。
  • 于 Android 上、点击 三點菜單 → 安裝應用程式。

🦊 Firefox

  • Firefox 當前不支持原生 PWA 安裝。
  • 可使用 PWAsForFirefox 插件插件實現桌面支持。

🧑‍💻 Edge（Windows・macOS・Android）

  • 訪問 PWA 網站。
  • 點選地址欄中之 安裝圖示、或進入 設定 → 應用程式 → 將此網站安装爲應用程式。
  • Android 上之安裝方式與 Chrome 相同。

🍎 Safari（iOS・macOS）

  • 在 iOS 上：點選 分享 圖示 → 添加至主熒幕。
  • macOS 上：Safari 不支援原生 PWA 安裝。
  • 建议使用 Chrome 或 Edge 瀏覽器以獲得完整支援。

📱 Android（所有瀏覽器）

  • 大多數現代瀏覽器均支援 PWA 安裝。
  • 于瀏覽器菜單中査找 安裝 或 添加至主熒幕 選項。

🧩 技术提示

  • PWA 必須通過 HTTPS 提供服務、並包含有效的 manifest.json 文件。
  • 可通過 Chrome DevTools → Application 標籤頁測試可安裝性。
  • 安裝後之 PWA 可像原生應用一樣從操作系統中啓動。

想瞭解更多？請訪問 Google 之 PWA 指南。

💚 安裝微信小程序

  • 于微信中開啓你希望使用之小程序。
  • 點選右上角之三點菜單。
  • 選擇添加至主熒幕或者添加至桌面。
  • 如果未有该選項或添加失敗、請檢査微信設定中是否啓用了桌面捷徑權限。
  • 成功添加後、小程序會像PWA一樣顯示于你之設備主熒幕上、方便快速訪問。
