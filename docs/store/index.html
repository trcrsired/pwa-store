How to Install PWAs

For an optimal experience on Android, we recommend using the Microsoft Edge browser. We advise against Firefox due to limited compatibility, and strongly discourage the use of locked-down operating systems like iOS, ChromeOS, Windows S mode and anything from Huawei, which restrict user freedom and app flexibility.

If you're browsing on Android, consider installing the NoForceDownload Chromium extension. Many websites aggressively prompt mobile users to download apps, and this extension helps block the most intrusive ones.
